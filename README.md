# Bank Management System (Core PHP)

This is a simple Bank Management System developed using Core PHP and MySQL.  
The application allows an admin to manage customer accounts and enables customers to securely view their own profile details.

## Features

### Admin Module
- Admin login authentication
- Add new bank customers
- Upload customer documents:
  - 3 photographs
  - Aadhar card
  - PAN card
- View all customer profiles
- Edit customer details
- Delete customer records
- Search customers by name or phone number

### Customer Module
- Customer login
- View personal profile only
- View uploaded documents (photos, Aadhar, PAN)

---

## Tech Stack

- Core PHP
- MySQL
- HTML5
- CSS3
- JavaScript (basic)
- XAMPP (Local Server)

## Database Structure

### Tables Used
- `users` – Stores admin and customer login credentials
- `customers` – Stores customer personal and document details
